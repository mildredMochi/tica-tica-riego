# 🌱 GUÍA DE INSTALACIÓN — Sistema de Riego Tica-Tica
# ══════════════════════════════════════════════════════

## PASO 1 — Instalar Streamlit en tu laptop (una sola vez)

Abre tu terminal/cmd y escribe:

    pip install streamlit supabase pandas

---

## PASO 2 — Copiar los archivos

Copia estos archivos a una carpeta nueva en tu laptop:

    📁 tica_tica_riego/
    ├── app.py
    ├── requirements.txt
    └── .streamlit/
        └── config.toml

---

## PASO 3 — Probar en tu laptop

En la terminal, entra a la carpeta y ejecuta:

    cd tica_tica_riego
    streamlit run app.py

Se abrirá automáticamente en tu navegador en:
    http://localhost:8501

---

## PASO 4 — Publicar gratis en Streamlit Cloud (para el celular)

1. Crea cuenta gratis en: https://github.com  (si no tienes)
2. Sube tu carpeta a GitHub (nuevo repositorio)
3. Ve a: https://share.streamlit.io
4. Conéctalo con tu GitHub
5. Selecciona tu repositorio y el archivo app.py
6. ¡Listo! Te da una URL pública como:
       https://tica-tica-riego.streamlit.app

---

## PASO 5 — Instalar en el celular como app

1. Abre la URL en Chrome del celular
2. Toca los 3 puntitos (menú)
3. "Agregar a pantalla de inicio"
4. ¡Se instala como app! 📱

## HORARIOS DE DATOS

El sistema muestra datos registrados a las:
  • 8:00 am
  • 12:00 pm
  • 6:00 pm
  • 10:00 pm

La laptop debe estar encendida en esos horarios.
Después de las 10pm puedes apagarla.
El celular puede ver los datos guardados en Supabase en cualquier momento. ✅

---

## SOPORTE

Si algo no funciona, verifica:
1. La laptop tiene internet
2. Supabase está conectado (URL y KEY correctas en app.py)
3. El ESP32 está enviando datos a la hora correcta
